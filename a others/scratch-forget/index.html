Structure

The structure of a html element consists of:
  <tag attributeName=" attributeValue">insert your text here</tag>

The structure of a html page) consists of:
  html,head, and body
where in head, you can include the page title (shown in the window tab)

Lists

Ordered Lists:

  1. list item 1
  2. list item 2
  3. list item 3

Unordered Lists:

  • list item 1
  • list item 2
  • list item 3

Definition Lists:

definition term 1
definition definition 1
definition term 2
definition definition 2
definition term 3
definition definition 3

Nested Lists:

Ordered-nested

  1. list item 1
    1. list item 1.1
    2. list item 1.2
  2. list item 2
    • list item 2.1
    • list item 2.2
  3. list item 3
    definition term1
    definition definition 1
    definition definition 2

Unordered-nested

  • list item 1
    1. list item 1.1
      • list item 2.1.1
    2. list item 1.2
  • list item 2
    • list item 2.1
      • list item 2.1.1
    • list item 2.2
  • list item 3
    definition term1
    definition definition 1
    definition definition 2
